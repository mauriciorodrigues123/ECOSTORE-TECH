🌱 ECOSTORE TECH
📖 Sobre o Projeto

A ECOSTORE TECH é um e-commerce de tecnologia com foco em sustentabilidade e responsabilidade social. O projeto busca reduzir os impactos ambientais causados pelo descarte incorreto de eletrônicos e pelo excesso de embalagens.

A proposta une tecnologia, inovação e consciência ambiental, mostrando que é possível criar um negócio sustentável e competitivo ao mesmo tempo.

♻️ Principais Soluções
📦 Uso de embalagens recicláveis e biodegradáveis
🔄 Logística reversa para descarte correto de eletrônicos antigos
🌱 Incentivo ao consumo consciente
💻 Informações sobre os impactos ambientais dos produtos
🎯 Objetivo

Promover um mercado digital mais responsável, incentivando práticas sustentáveis no setor de tecnologia.
